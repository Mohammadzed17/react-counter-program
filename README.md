# React Counter Program

A simple counter application built using React and the useState hook.

## 🚀 Features

- Increment count
- Decrement count
- Reset count to zero
- Functional state updates (safe updater function)
- Clean component-based structure

## 🛠️ Built With

- React (Vite)
- JavaScript (ES6+)
- useState Hook

## 📂 Project Structure


react-counter-program/
│
├── public/
├── src/
│ ├── Counter.jsx
│ └── main.jsx
├── package.json
├── vite.config.js
└── index.html


## ▶️ How to Run Locally

1. Clone the repository


git clone https://github.com/Mohammadzed17/react-counter-program.git


2. Navigate to the project folder


cd react-counter-program


3. Install dependencies


npm install


4. Start development server


npm run dev
